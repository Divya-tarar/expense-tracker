# 💰 Expense Tracker App

A modern Expense Tracker built using React and Vite.

## 🚀 Features
- Add & delete expenses
- Categorize expenses
- Filter by category
- Income & Expense calculation
- Data stored in Local Storage
- Routing using React Router

## 🛠 Tech Stack
- React.js
- Vite
- CSS
